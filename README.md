Ekta Uniyal Portfolio

How to use:
1. Download this folder.
2. Replace public/ekta-cartoon.png with your image  if you want the same hero image.
3. Place your resume file .
4. Run:
   npm install
   npm start

This is a minimal React project (create-react-app style). If you want a zipped build ready for Netlify/GitHub Pages, ask me and I can provide instructions.
